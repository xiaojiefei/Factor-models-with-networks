# 论文方法论笔记

**论文**：Bonaccolto et al. (2019)  
**完整标题**：Estimation and model-based combination of causality networks among large US banks and insurance companies  
**期刊**：Journal of Empirical Finance, Vol. 54, pp. 1–21  
**PDF**：`文献/Bonaccolto 等 - 2019 - Estimation...pdf`

---

## 核心研究问题

1. 不同因果检验方法（Granger vs 分位数因果）产生的网络结构有何不同？
2. 如何将多种因果网络（多层/multiplex）进行最优组合？
3. 分位数因果网络能否更好地捕捉金融风险的传播渠道？

**两大贡献**：
- **贡献一**：提出基于线性因子模型的多层网络组合方法（model-based combination）
- **贡献二**：将分位数因果检验引入金融网络估计，捕捉风险维度

---

| 字段 | 内容 |

## 核心方法论

### 1. 因果网络估计方法（4种 → 构成多层网络的各层）

| 方法 | 代号 | 检测的因果类型 | 关键参考文献 |
|------|------|--------------|-------------|
| **Granger 因果** | GR | 均值因果（线性） | Billio et al. (2012) |
| **基础分位数回归** | QB | 分位数因果（线性、全分布影响） | Koenker & Bassett (1978) |
| **Quantile-on-Quantile** | Qo | 分位数因果（线性、解释变量的局部分布） | Sim & Zhou (2015) |
| **非参数分位数因果** | QN | 分位数因果（非线性、非参数核方法） | Jeong et al. (2012) |

**网络构建流程**：
- 对每一对资产 (i, j)，进行因果检验（lag=1）
- H₀：i 对 j 无因果关系
- 拒绝 H₀ (5%水平) → w_{i,j} = 1；接受 → w_{i,j} = 0
- 结果：K×K 有向无权邻接矩阵 W（排除自环：w_{i,i}=0）
- 行标准化后用于因子模型

**分位数水平**：τ = 0.1, 0.5, 0.9
**统计推断**：xy-pair bootstrap, 5000 次重抽样
**Qo 带宽**：h = 0.1（plug-in）
**QN 带宽**：LSCV 方法 (Rudemo 1982, Bowman 1984)
**核函数**：Gaussian

### 2. 模型组合方法：带多层空间依赖的线性因子模型

**结构方程 (Eq. 4)**：
```
A · R_t = E[R_t] + β · F_t + η_t

A = I - Λ · (Σ_{j=1}^{d} δ_j · W_j)
```

**参数说明**：
- `R_t` (K×1)：资产收益向量
- `F_t`：公共因子（FF4因子）
- `W_j` (K×K)：第j层行标准化邻接矩阵，j=1,...,d
- `δ_j`：网络层权重，δ_j ≥ 0, Σδ_j = 1（凸组合）
- `Λ = diag(ρ_1,...,ρ_K)`：每个资产对复合网络的异质反应系数
- `β` (K×F)：因子暴露矩阵
- `η_t`：结构残差，Σ_η 为对角阵

**复合网络 (Eq. 6)**：
```
W* = Σ_{j=1}^{d} δ_j · W_j
```

**估计方法**：集中最大似然法（Concentrated MLE）
1. 给定 A → OLS 估计 β 和 Σ_η
2. 将 β、Σ_η 代回 → 关于 (δ_j, ρ_i) 的集中似然
3. 约束优化求解 δ_j 和 Λ

**识别假设**：
- A1：每层网络 W_j ≠ 0
- A2：W_i ≠ W_j, ∀i ≠ j
- A3：δ_j ≥ 0, Σδ_j = 1
- A4：W* 每行至少一个非零元素

### 3. 网络评估指标

| 指标 | 含义 |
|------|------|
| 密度 (Density) | 实际边数 / 最大可能边数 |
| 同配性 (Assortativity) | 高度数节点是否倾向连接高度数节点，范围(-1,1) |
| 特征向量中心性 (Eigenvector Centrality) | 节点重要性（考虑邻居的重要性），取均值 |
| Jaccard 指数 | 两个网络的边集相似度，范围(0,100) |

---

## 关键公式

1. **Eq. 1** (结构方程): `A(R_t - E[R_t]) = β̄·F_t + η_t`
2. **Eq. 2** (简化方程): `R_t = E[R_t] + A⁻¹·β̄·F_t + A⁻¹·η_t`
3. **Eq. 3** (单参数多层): `A = I - Σ_{j=1}^{d} ρ_j·W_j`
4. **Eq. 4** (异质反应多层): `A = I - Λ·(Σ_{j=1}^{d} δ_j·W_j)`（核心模型）
5. **Eq. 6** (复合网络): `W* = Σ_{j=1}^{d} δ_j·W_j`
6. **Eq. 7** (边的定义): `w_{i,j} = 1 if H₀ rejected, 0 otherwise`

## 复现难点

1. **非参数分位数因果 (Jeong et al. 2012)**：涉及非参数条件分布估计和LSCV带宽选择，实现复杂度高
2. **Quantile-on-Quantile**：需要在解释变量的特定分位数邻域内做局部线性回归
3. **集中 MLE 优化**：约束优化（δ_j凸组合 + Λ对角 + A可逆性检查）
4. **计算量**：48个资产 → 48×47 = 2256 对因果检验 × 4种方法 × 3个分位数
5. **Bootstrap 计算**：每对资产5000次重采样，总计约 33.8M 次回归
