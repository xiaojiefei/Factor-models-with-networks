# 基于 Bonaccolto (2019) 的创新方向

> 在 Bonaccolto et al. (2019) 多层因果网络因子模型框架上，可行的研究创新点。

---

## 一、混合"金融网络"与"实体网络" ⭐⭐⭐⭐⭐

### 核心思想

Bonaccolto 的多层网络**全部来自同一组收益率数据**（只是因果检验方法不同：GR, QB, Qo, QN）。创新点是加入**不同数据来源**的网络层：

- $W_1$ = DY 收益率溢出（金融渠道）
- $W_2$ = DY 波动率溢出（风险渠道）
- $W_3$ = **投入产出表**（实体经济渠道）

### 研究问题

$\delta_3$ 的大小直接回答：**实体经济关联在解释行业收益率共动中有多重要？**

### 数据来源

投入产出表数据国家统计局有公开的（2017/2018/2020年版本），可构造行业间中间品流动矩阵，行标准化后作为 $W_3$。

### 为什么是创新

- 打破了"所有网络来自同源数据"的局限
- 连接了金融网络文献和实体经济网络文献
- 让 $\delta_j$ 的比较有更强的经济含义（金融渠道 vs 实体渠道）

---

## 二、$\delta_j(t)$ 的驱动因素分析 ⭐⭐⭐⭐⭐

### 核心思想

Bonaccolto 只估了 $\delta$，没问"**为什么 $\delta$ 会变**"。第二阶段回归：

$$
\delta_j(t) = \gamma_0 + \gamma_1 \cdot VIX_t + \gamma_2 \cdot GPR_t + \gamma_3 \cdot MPU_t + \varepsilon_t
$$

### 候选解释变量

| 变量          | 含义                 | 数据来源             |
| ------------- | -------------------- | -------------------- |
| VIX           | 全球风险情绪         | CBOE                 |
| GPR           | 地缘政治风险指数     | Caldara & Iacoviello |
| MPU           | 货币政策不确定性     | Baker, Bloom & Davis |
| EPU_CN        | 中国经济政策不确定性 | 同上（中国版）       |
| Credit Spread | 信用利差             | Wind                 |

### 研究问题

- 什么宏观环境下，波动率渠道比收益率渠道更重要？
- 危机期间 $\delta_{vol}$ 是否显著上升？（风险传导渠道在压力下被激活）
- 实体渠道 $\delta_{IO}$ 在何时主导？

### 为什么是创新

- 从描述性分析（画 $\delta$ 时间图）升级为因果性分析（为什么 $\delta$ 变化）
- 政策含义更强：告诉监管者在什么条件下需要关注哪种传导渠道

---

## 三、韧性视角重构 ⭐⭐⭐⭐

### 核心思想

Bonaccolto 关注的是**风险传染**（$\rho$ 越大越危险）。翻转视角看**韧性**：

- 不只看"冲击传多大"，还看"恢复多快"
- 识别"韧性枢纽"行业：在冲击中反而稳定网络的节点

### 可操作定义

1. **韧性指标1**：冲击后收益率回复速度（半衰期）
2. **韧性指标2**：特定冲击事件前后 $\rho_i$ 的变化幅度（小变化 = 高韧性）
3. **韧性指标3**：网络中心性在冲击期的稳定性（中心性不跳 = 结构韧性）

### 对比不同冲击

| 事件              | 时间          | 冲击类型     |
| ----------------- | ------------- | ------------ |
| 2008 全球金融危机 | 2008Q3-2009Q1 | 金融冲击     |
| 2015 中国股灾     | 2015Q2-Q3     | 国内金融冲击 |
| 2018 中美贸易摩擦 | 2018Q2-Q4     | 地缘政治冲击 |
| COVID-19          | 2020Q1        | 公共卫生冲击 |
| 俄乌冲突          | 2022Q1-Q2     | 地缘政治冲击 |

### 为什么是创新

- "韧性"是宏观金融研究的新热点（Brunnermeier, 2024）
- 与项目申报书的"多种外部冲击下的韧性"主题完全匹配
- 提供了叙事角度，让模型结果更有政策含义

## 四、非对称网络效应 ⭐⭐⭐

### 核心思想

Bonaccolto 假设 $\rho_i$ 对正冲击和负冲击一样大。但现实中恐慌比乐观传播得更快。

### 根本困难：$(I - \rho W)^{-1}$ 的闭式解丢失

直接用 $\rho^+$ 和 $\rho^-$ 区分正负冲击会引入 $\max(\cdot,0)$ 非线性：

$$
\tilde{R}_{i,t} = \rho_i^+ \sum_j W_{ij} \max(\tilde{R}_{j,t}, 0) + \rho_i^- \sum_j W_{ij} \min(\tilde{R}_{j,t}, 0) + \bar{\beta}_i F_t + \eta_{i,t}
$$

问题：$\tilde{R}_{j,t}$ 本身是内生的（出现在等式两边），$\max/\min$ 使其变成非线性不动点问题。三个连锁后果：

| 原模型                      | 非对称模型                               |
| --------------------------- | ---------------------------------------- |
| $A = I - \rho W$ 常数矩阵 | $A_t$ 每个时点不同（依赖于收益率符号） |
| Jacobian = $T \ln|A         | $，一次算完                              |
| 集中MLE：OLS 解$\beta$    | OLS 不能直接做（分段线性）               |

### 可行替代方案

| 方案                   | 难度     | 保留$(I-\rho W)^{-1}$？ | 定位         |
| ---------------------- | -------- | ------------------------- | ------------ |
| **外生体制切换** | ⭐⭐     | ✅ 保留                   | 正文扩展分析 |
| 分位数 SAR             | ⭐⭐⭐⭐ | ❌ 需重写                 | 另一篇论文   |
| 两步法残差检验         | ⭐       | ✅ 保留                   | 稳健性检验   |

### 推荐：外生体制切换（方案1）

用外部指标定义牛/熊体制（如 VIX 阈值），在每个体制内模型仍然是标准线性 SAR：

- 熊市：$A^{bear} = I - \Lambda^{bear} W$ → 估出 $\rho_i^{bear}$
- 牛市：$A^{bull} = I - \Lambda^{bull} W$ → 估出 $\rho_i^{bull}$

比较 $\rho^{bear}$ vs $\rho^{bull}$ 即可验证非对称性，**无需修改估计框架**。

## 五、样本外预测 + 组合构建 ⭐⭐⭐

### 核心思想

Bonaccolto 只做样本内估计。扩展到样本外：

1. 用前 $T-h$ 期估计模型 → 预测后 $h$ 期收益率
2. 基于网络中心性构建投资组合（做多低中心性/高韧性行业）
3. 与传统等权/市值加权组合比较 Sharpe ratio

### 为什么是创新

从"学术"走向"应用"，增加实用价值。但创新度相对较低（许多网络论文都做这个）。

## DY 溢出网络作为 W 的合理性

### 波动率溢出 → 收益率的经济机制

DY 波动率溢出网络用于收益率模型的经济逻辑：

1. **风险重定价**：$j$ 波动率↑ → 市场认为 $i$ 风险也↑ → $i$ 的风险溢价变化 → $i$ 收益率变化
2. **杠杆/保证金螺旋**：$j$ 波动率飙升 → 保证金追缴 → 被迫卖出 $i$ → $i$ 收益率↓
3. **避险迁移**：高风险资产波动率↑ → 资金涌向安全资产 → 安全资产收益率↑

### 与 Bonaccolto 原文的区别

| 维度         | Bonaccolto (2019)          | 本项目                     |
| ------------ | -------------------------- | -------------------------- |
| 网络构造方法 | 因果检验（GR, QB, Qo, QN） | DY 溢出指数                |
| 网络稀疏性   | 稀疏（p值筛选，有零元素）  | 稠密（FEVD全非零）         |
| 术语         | "因果网络"                 | "信息溢出网络"（审稿时用） |
| 网络数据来源 | 全部来自同一组收益率       | 可混合不同数据源           |

## 六、贝叶斯估计替代 MLE ⭐⭐⭐⭐

### 核心思想

MLE 给点估计，不知道 $\hat{\rho}_i = 0.3$ 的不确定性多大。贝叶斯 MCMC 输出后验分布。

### 做法

- $\rho_i$ 先验：$\rho_i \sim \text{Beta}(2, 5)$（倾向于小值但允许大值）
- $\delta$ 先验：$(\delta_1, \delta_2) \sim \text{Dir}(\alpha_1, \alpha_2)$（自动满足凸组合约束）
- 输出：后验分布 + 95% 可信区间 + Bayes Factor 模型比较

### 价值

- 可信区间直接判断显著性（区间是否包含0）
- 小样本（1290周）下比 MLE 更可靠
- Bayes Factor 可以比较嵌套模型（2层 vs 3层）

### 难度

中等。`pymc` 或 `numpyro` 可用，但需自写似然函数，调试 MCMC 收敛需要经验。

## 七、状态空间模型替代滚动窗口 ⭐⭐⭐⭐

### 核心思想

滚动窗口（window=52）有三个硬伤：
1. 窗口大小人为选择（为什么52不是40？）
2. 窗口内假设参数不变（52周内可能已发生结构变化）
3. 浪费数据（窗口越大，可用滚动点越少）

### 做法

把 $\rho_i(t)$ 和 $\delta_j(t)$ 建模为随机游走：

$$\rho_i(t) = \rho_i(t-1) + v_{i,t}, \quad v_{i,t} \sim N(0, \sigma_v^2)$$

用 Kalman Filter 或粒子滤波估计。

### 价值

- 不需要选窗口大小
- 每个时点都有估计值（不是每4周一个）
- 更高效利用数据

### 难度

高。ρ 进入似然是非线性的 → 需要粒子滤波或扩展卡尔曼滤波，计算量大。

## 八、尾部相依网络 ⭐⭐⭐⭐

### 核心思想

DY 溢出捕捉的是**均值/方差层面**的关联。但金融风险管理最关心**极端事件共动**——两个行业正常时不相关，崩盘时同时暴跌。

### 做法

构建第三层网络 $W_3$ = 尾部相依网络：
- CoVaR 网络：$W_{ij}$ = 行业 j 处于极端状态时行业 i 的条件风险值
- 或 Copula 尾部相依系数：$\lambda_{ij} = \lim_{q \to 0} P(R_i < F_i^{-1}(q) \mid R_j < F_j^{-1}(q))$

### 三层结构

| 层 | 网络 | 捕捉什么 |
|---|---|---|
| $W_1$ | DY 收益率溢出 | 正常时期信息传导 |
| $W_2$ | DY 波动率溢出 | 不确定性传导 |
| $W_3$ | 尾部相依 | 极端事件共动 |

### 价值

$\delta_3$ 直接告诉你：**极端共动在解释收益率关联中有多重要？** 如果危机期间 $\delta_3$ 上升，说明尾部渠道在压力下被激活。

## 九、嵌套模型检验 ⭐⭐⭐⭐⭐（必须做）

### 核心思想

"多层网络比单层好"需要**正式统计检验**证明，不能只靠直觉。

### 做法

构建一组嵌套模型：

| 模型 | 设定 | 参数数 |
|------|------|--------|
| M0（基准） | 无网络：$A = I$（标准因子模型） | $K \cdot n_f + K$ |
| M1 | 单层：$A = I - \Lambda W_1$ | $M0 + K$ |
| M2 | 单层：$A = I - \Lambda W_2$ | $M0 + K$ |
| M3 | 两层：$A = I - \Lambda(\delta_1 W_1 + \delta_2 W_2)$ | $M0 + K + 1$ |
| M4 | 三层（加 I/O 表） | $M0 + K + 2$ |

用**似然比检验**（LR test）逐层比较：
- M0 vs M1：网络到底有没有用？（$H_0: \rho_i = 0 \ \forall i$）
- M1 vs M3：第二层网络有没有增量信息？
- M3 vs M4：投入产出表有没有独立贡献？

### 价值

- 审稿人最看重——**统计检验证明多层显著更好**，不是"我觉得"
- 实现极简单（已有MLE，记下每个模型的 log-likelihood 即可）
- AIC/BIC 也可以算，处理过拟合担忧

## 十、网络内生性处理 ⭐⭐⭐⭐⭐（必须做）

### 核心思想

模型假设 W 是**外生的**（先从 DY 估出来，再插入因子模型）。但实际上：
- 收益率 $R_t$ 影响 DY 网络 $W_t$（W 就是从 R 算出来的）
- $W_t$ 又影响 $R_t$（通过模型）
- 这是**同时性偏误**（simultaneity bias）

### 三种做法

| 方案 | 难度 | 说明 |
|------|------|------|
| 用滞后网络 $W_{t-1}$ | ⭐（改一行代码） | 上一期网络不受本期收益率影响 |
| 两阶段估计（2SLS） | ⭐⭐⭐ | 用工具变量 |
| 联合估计 W + 因子模型 | ⭐⭐⭐⭐⭐ | 计算量极大 |

### 价值

用 $W_{t-1}$ 只需改一行代码，但显著增强因果识别可信度。审稿人几乎必问这个问题。

## 十一、多重网络度量（Multiplex metrics）⭐⭐⭐

### 核心思想

现在只算了复合网络 $W^* = \sum \delta_j W_j$ 的特征向量中心性。多层网络有更丰富的结构信息。

### 新度量

| 度量 | 含义 | 经济解释 |
|------|------|---------|
| 多重参与系数 | 行业在各层的中心性是否一致 | 高 = 在所有渠道都重要；低 = 只在特定渠道重要 |
| 层间度相关 | 在 $W_1$ 中重要的行业在 $W_2$ 中也重要吗？ | 高 = 渠道冗余；低 = 渠道互补 |
| 多重社区检测 | 哪些行业跨层形成稳定的"簇" | 识别系统性重要的行业群组 |

### 价值

让论文的网络分析部分更丰富，超越"算一个中心性排名"的层次。

## 遗漏变量问题（OVB）：遗漏关联矩阵

### 问题本质

如果真实模型有 d=5 层网络但只放了 d=2 层，遗漏的网络层导致：

| 标准回归 OVB | 网络因子模型 OVB |
|-------------|-----------------|
| 遗漏 $X_2$ | 遗漏 $W_3$（如投入产出表） |
| $\hat{\beta}_1$ 有偏 | $\hat{\delta}_1, \hat{\delta}_2, \hat{\rho}_i$ 都有偏 |
| 偏误条件：$X_2$ 与 $X_1$ 相关 | 偏误条件：$W_3$ 与 $W_1, W_2$ 相关 |

### 遗漏的 $\delta_3 W_3$ 去了两个地方

1. **被已有的 $W_1, W_2$ 吸收**：如果 $W_3$（投入产出）和 $W_1$（收益率溢出）相关，$\tilde{\delta}_1$ 偏大——不只反映金融渠道，还代理了实体渠道
2. **漏进残差 $\eta_t$**：$W_3$ 中与 $W_1, W_2$ 不相关的部分进入结构残差，导致 $\eta_t$ 不再独立——违反核心假设 A2

### 严重程度判断

| 情况 | 严重程度 |
|------|---------|
| 遗漏的 $W_3$ 与已有 $W_1, W_2$ 高度共线 | 低——已有网络已代理大部分效应 |
| 遗漏的 $W_3$ 与已有网络正交 | 高——完全没捕捉到的渠道 |
| 遗漏渠道效应本身很小（$\delta_3 \approx 0$） | 低——即使遗漏影响也小 |

### 论文中如何处理

1. **正文**：加入 $W_3$（投入产出表），直接减少遗漏（方向一）
2. **稳健性**：比较 2层 vs 3层模型的 $\hat{\rho}_i$ 是否显著变化（方向九）
3. **局限性讨论**：诚实说明"仍可能遗漏其他渠道（地理邻近、供应链等），但框架具有可扩展性"

### 与方向一的关系

OVB 问题正是方向一（混合金融+实体网络）的**立论基础**：
> 现有文献只用金融市场数据构建网络，可能遗漏实体经济关联渠道。如果估计发现 $\hat{\delta}_3 > 0$ 且显著，就证明了只用金融网络的模型有偏。
